How to input formula

The description inputting formula

Format

Enter half-width characters

  • Enter 1+1 instead of １＋１

Usable characters are +-*/(), integers and real numbers

  • Enter 1*1 for 1×1
  • Enter 1/1 for 1÷1
  • Integer is 1, -1, etc.
  • Real is 0.1, 10.001, etc.

Use . as decimal point

  • Use 0.1 instead of 0,1

Calculations that use negative numbers other than subtraction should be enclosed in ()

  • 2*3*-1 should be 2*3*(-1)

Specification

The length of the formula you can enter is up to 64 characters, including spaces

Whitespace is ignored

  • 1 + 2 + 3 is the same as 1+2+3

If you enter only a numerical value, it will be output as the calculation result

  • Enter 2 in the formula, it will output 2

Invalid formula

If the expression is as follows, an error will be shown

( is preceded by a number or )

  • 6/2(1+2), (6/2)(1+2)

) followed by a number or (

  • (1+2)3, (6/2)(1+2)

() is not paired

  • (1+2, 1+2)

The expression ends with an operator

  • 1+2+

There is no operator between the numbers

  • 1 2 3
  • 1 2 + 3

Dividing by 0

  • 1/0

Result format

The calculation result will be the calculated value or an error message

  • Check Error description for more information about the error

The calculation result is displayed to the 10th decimal place, and any numbers smaller than that are rounded up or off

  • 1/3 = 0.3333333333
  • 1/2048 = 0.0004882813

Division may result in rounding to the nearest (11 - number of integer digits) decimal place

  • The number of integer digits and decimal places may be adjusted to 10 digits
  • 65535/1234 = 53.10777958

Error description

If the formula cannot be interpreted, an error message is returned instead of the calculation result

Levels of error messages

Show 3 levels of error messages (Tiny, Medium, Full) based on the verbosity and specificity of the error
